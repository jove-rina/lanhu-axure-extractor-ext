WeChat Integration

Connect Hermes Agent to personal WeChat — message send/receive, multi-profile isolation, and troubleshooting.

🏷️ v1.9.0 🔧 Hermes Agent ✓ iLink Bot API
⚡ 快速安装
# 添加 tap 源
hermes skills tap add jove-rina/jove-hermes-skills

# 安装 WeChat 集成技能
hermes skills install jove-rina/jove-hermes-skills/hermes-wechat-integration
特性

核心特性

六大特性让 WeChat 集成安全、高效、易于管理

🎯

QR Code Auth

Official iLink Bot API, no reverse engineering. QR proxy protocol: AI presents URL, user scans.

🛡️

Two-Gate Auth

Policy gate (env vars) + Routing gate (channel_directory.json), diagnosis matrix pinpoints issues

⚡

context_token Awareness

Prevents silent message drops: send_message success ≠ delivery. Live connection vs throwaway adapter.

🔒

Multi-Profile Isolation

Each Hermes profile with its own WeChat account, zero interference between bots

🌐

Cross-Platform

Linux / macOS / Windows Git Bash. Same commands everywhere.

📦

Token-Efficient

Core SKILL.md is 4.3K tokens. FAQ and placeholder table lazy-loaded. Saves 64% tokens vs original.

方案

问题 vs 解决方案

WeChat 没有官方个人 bot API，逆向工程有封号风险。这个技能用官方渠道解决了问题。

✕ 问题

WeChat 没有官方 Bot API

微信没有面向个人的官方 Bot API。逆向工程（如 ItChat、WeChatFerry）违反用户协议，轻则封号，重则法律风险。现有第三方方案部署复杂、维护困难，且缺乏 AI-native 协议支持。

✓ 解决方案

官方 iLink Bot API + AI-Native 协议

此 Hermes Agent 技能使用官方 iLink Bot API，通过 QR 码配对安全连接个人微信。AI-native 协议处理 AI 无法完成的操作（扫码、发送测试消息等）。双门授权机制提供精细的访问控制，context_token 机制防止静默投递失败。无需逆向工程，安全合规。

流程

快速上手

5 步完成从安装到验证

1
安装依赖
安装 Python 依赖包 aiohttp 和 cryptography
pip install aiohttp cryptography
2
初始化网关
运行 hermes gateway setup，AI 会展示 QR URL，用户手机扫码完成配对
hermes gateway setup
3
提取 Account ID 并配置 .env
从 setup 输出中提取 account_id，写入 HERMES_HOME/.env 文件
echo "WEIXIN_ACCOUNT_ID=<account_id>" >> $HERMES_HOME/.env
4
配置授权策略
设置 DM 策略和允许用户列表，控制谁能与 AI 对话
echo "WEIXIN_DM_POLICY=allowlist" >> $HERMES_HOME/.env
echo "WEIXIN_ALLOWED_USERS=<user_id>" >> $HERMES_HOME/.env
5
启动并验证
启动网关，等待 3 秒后检查日志确认连接正常
hermes gateway start
sleep 3 && grep "gateway" $HERMES_HOME/logs/*.log
